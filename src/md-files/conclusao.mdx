- Foi gerado um script chamado merge-dependencies.js com IA para pegar as dependencias do SDS e jogar em cima desse repositório. Por conta disso, o package.json está completamente poluído.
- No geral a ferramenta funciona bem, mas é preciso estar corretamente estruturado para que se ganhe o benefício de gerar telas rapidamente a partir de componentes previamente cadastrados.
