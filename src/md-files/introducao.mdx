<TypographyP>Este repositório foi criado para demonstrar como utilizar o <InlineCode>Figma Code Connect</InlineCode> com o <InlineCode>SDS</InlineCode> (Sample Design System). Abaixo, explicamos o que são essas ferramentas e como elas podem ser usadas juntas.</TypographyP>

<Separator />

<TypographyP>O <InlineCode>Figma Code Connect</InlineCode> é uma ferramenta que conecta projetos de design criados no Figma diretamente ao código. Ele permite que designers e desenvolvedores colaborem de forma mais eficiente, reduzindo o esforço manual necessário para converter designs em código utilizável. Para mais informações, acesse o repositório oficial no GitHub: 
<TypographyLink to="https://github.com/figma/code-connect" target="_blank" rel="noopener noreferrer">Figma Code Connect</TypographyLink>.</TypographyP>

<TypographyP>O <InlineCode>SDS</InlineCode> (Sample Design System) é um sistema de design base que demonstra como usar as Variáveis, Estilos, Componentes e o <InlineCode>Figma Code Connect</InlineCode> para criar um sistema de design responsivo completo integrado com um código React. Ele não apenas conecta design e desenvolvimento, mas também oferece boas práticas para preencher as lacunas entre esses dois mundos. Para saber mais, visite o repositório oficial no GitHub: 
<TypographyLink to="https://github.com/figma/sds" target="_blank" rel="noopener noreferrer">Figma SDS</TypographyLink>.</TypographyP>

<Separator />

<TypographyP>O que faremos aqui é usar os componentes gerados pelo <InlineCode>SDS</InlineCode> e previamente conectados a uma cópia do arquivo de Design System do <InlineCode>SDS</InlineCode> no Figma em nosso repositório para importar telas diretamente com pouca ou nenhuma necessidade de codificação.</TypographyP>
