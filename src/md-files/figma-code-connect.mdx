<TypographyP>As instruções abaixo foram retiradas do README do SDS. Serão feitas elas aqui com screenshots para facilitar o entendimento e reprodução.</TypographyP>

<TypographyP>Os passos originais do repositório do SDS são:</TypographyP>

<CodeBlock language="shell">{`1) Duplicate the Figma Community File
2) Clone this repo
3) Update urls in figma.config.json to point to your Figma file
   Note: the file keys (eg. J0KLPKXiONDRssXD1AX9Oi) should be the only change in the urls unless you're creating new components, detaching and recreating.
4) Create and set your Figma Auth Token
5) At that point, npx figma connect publish should work and your new file should have Code Connect.`}
</CodeBlock>

<SubtitleMd>Passo 1</SubtitleMd>

<TypographyP>Para o ```item 1```, simplesmente acessamos o <TypographyLink to="https://www.figma.com/community/file/1380235722331273046/simple-design-system" target='_blank'>link do arquivo de SDS no Figma</TypographyLink> e criamos uma cópia na nossa conta como na imagem abaixo.</TypographyP>

<div className="mt-6">
![Criar cópia do arquivo de design system do SDS](/images/figma-sds-copy.png)
</div>

<SubtitleMd>Passo 2</SubtitleMd>

<TypographyP>Após isso, clonamos o <TypographyLink to="https://github.com/figma/sds" target='_blank'>repositório do SDS</TypographyLink> para concluir o ```item 2```.</TypographyP>

<CodeBlock>git clone https://github.com/figma/sds</CodeBlock>

<SubtitleMd>Passo 3</SubtitleMd>

<TypographyP>Dentro do repositório SDS clonado, devemos substituir o ID do arquivo de Design System original pela cópia no arquivo **figma.config.json**. No exemplo, o original era **J0KLPKXiONDRssXD1AX9Oi** e foi sobreescrito por **HtE2Ja2xxQegkAIXrLQLGP** que é o ID da nossa cópia usada (abaixo demonstramos como obter essa informação).</TypographyP>

<div className="mt-6">
![Replace de J0KLPKXiONDRssXD1AX9Oi pelo id do seu arquivo cópia](/images/sds-step3-1.png)
</div>

<SubtitleMd>Passo 4</SubtitleMd>

<TypographyP>É necessária uma chave de API com a permissão do code connect. Atualmente essa chave está apenas disponível para contas no plano organization ou business.</TypographyP>

<TypographyP>Essa chave deve ser gerada nas configurações de usuário:</TypographyP>

<div className="mt-6">
![Abrindo as configurações de usuário](/images/sds-step4-1.png)

<div className="mt-6">
![Marcar a opção da geração do token com permissão de escrita no code connect](/images/sds-step4-3.png)

<SubtitleMd>Passo 5</SubtitleMd>

<TypographyP>Agora podemos executar o comando de sincronização do repositório do SDS com o nosso arquivo de design system copiado dentro do figma.</TypographyP>

<CodeBlock language="shell">{`npx figma connect publish --token <TOKEN DO FIGMA GERADO NO PASSO 4>`}</CodeBlock>
